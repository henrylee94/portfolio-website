# Portfolio Website

[![Deployed on Vercel](https://img.shields.io/badge/Deployed_on-Vercel-black?style=flat&logo=vercel)](https://portfolio-website-three-henna-93.vercel.app)

Personal portfolio site for Henry Lee — Backend Team Lead.

**Live:** [portfolio-website-three-henna-93.vercel.app](https://portfolio-website-three-henna-93.vercel.app)

## Overview

A minimal, dark-themed single-page portfolio showcasing backend engineering experience: system design, microservices migration, and production-scale platform work.

## Tech Stack

- HTML5 / CSS3 (custom properties, dark theme)
- Vanilla JavaScript (animations, i18n)
- No build tools — zero dependencies

## Structure

```
index.html          — Main page
css/styles.css      — Styles
js/main.js          — Core logic
js/animations.js    — Scroll & transition animations
js/i18n.js          — Internationalization
```

## Local Development

```bash
# Any static server works
python3 -m http.server 8080
# Then open http://localhost:8080
```
